# Operational & Payments Analytics for Education Business

## Business Context
This project analyzes real operational and payments data from an education business.
The goal is to improve revenue visibility, attendance tracking and cash flow decisions.

## Dataset
Anonymized transactional and operational data:
- students
- lessons
- payments
- schedules

## Tools
- SQL
- Google Sheets / Excel
- Python (basic)
- GitHub

## Key Questions
- How much revenue should have been collected vs actually collected?
- Which clients generate the most value?
- Where do payment delays and gaps occur?
- What operational KPIs matter for daily decisions?

## Output
- SQL analysis
- Cleaned datasets
- KPI dashboards
- Business insights and recommendations
